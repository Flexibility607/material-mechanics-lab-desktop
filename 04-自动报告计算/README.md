# 材料力学七次实验自动报告计算

本目录把 `实验汇总/` 中原先分散的计算程序重新整理为一个统一入口。计算统一采用 **N–mm–MPa** 单位制，应变仪读数统一使用微应变（$10^{-6}$）。

## 一、自动生成报告前必须提供的数据

### 1. 所有报告都需要

| 类别 | 必填内容 | 用途 |
|---|---|---|
| 报告信息 | 实验名称、姓名、学号、班级、教师、同组者、日期 | 生成封面和报告头 |
| 试件信息 | 材料名称、截面形式、实际尺寸测量值 | 选择公式并计算 $A$、$I$、$W$ |
| 加载信息 | 初载荷、末载荷或各级载荷、重复次数、力臂/跨度 | 计算载荷增量、弯矩和扭矩 |
| 测量信息 | 各次仪器原始读数及单位 | 求平均值、增量、拟合和误差 |
| 测量约定 | 应变片方向、通道对应关系、桥路显示系数、正负方向 | 防止应变相差 2 倍或 4 倍、符号颠倒 |
| 异常说明 | 损坏通道、无效读数、加载波动等 | 自动跳过无效值并在报告中说明 |

材料牌号、给定弹性模量、泊松比、重力加速度等若已经固定在课程模板中，可作为配置默认值；若实验装置或试件发生变化，则必须重新提供。

### 2. 七次实验的最小必填量

| 实验 | 必须提供的原始量 | 程序自动计算 |
|---|---|---|
| 材料力学性能 | 各试件初始直径；低碳钢拉伸的 $F_s,F_b,l_0,l_1,d_1$；铸铁拉伸 $F_b$；压缩 $F_s/F_b$；扭转 $T_b$、扭转角；实验现象 | 截面积、$sigma_s$、$sigma_b$、$delta$、$psi$、$	au_b$ |
| 弹性常数 | 宽度和厚度的多次测量；每次加载的载荷序列；4 个应变通道读数；轴向/横向通道编号 | $A$、各级 $Delta\varepsilon$、$E_i$、$\mu_i$、平均 $E$、平均 $\mu$、线性拟合 |
| 切变模量 | 圆轴直径；力臂 $a$；标距 $L$；百分表臂长 $b$；各级载荷、百分表读数、半桥/全桥读数及显示系数 | $I_p$、$W_p$、$T$、$\varphi$、$\gamma$、两种方法的 $G$ |
| 直梁弯曲 | $E,\mu$；梁宽和梁高；载荷增量与加载间距；各 $y$ 坐标的纵向应变；上下表面横向应变；全桥读数和显示系数 | $I_z$、理论/实验应力、相对误差、实测 $\mu$、最大弯曲正应变 |
| 梁变形 | 简支梁 $E,L,b,h,\Delta P$；跨中挠度、转角百分表读数、测量臂长、互等读数；悬臂梁 $E,b,h,l_{12}$ 和两位置应变 | 理论挠度、理论转角及误差、位移互等差、悬臂梁测得质量 |
| 弯扭组合 | $E,\mu,d,\Delta F,L_M,L_T$；上下表面 $0^\circ,\pm45^\circ$ 应变花读数；弯矩半桥和扭矩全桥读数及显示系数 | 主应力、主方向、理论值和误差、实测弯矩、实测扭矩 |
| 偏心拉伸 | 截面 $h,b$；载荷增量；四分之一桥两侧应变；全桥 $2\varepsilon_F$；半桥 $2\varepsilon_M$ | $A$、$W_z$、$E$、最大应变/应力、偏心距 $e$ |

## 二、不能仅靠数值自动得到的内容

以下内容仍需实验者提供，或在报告模板中预先固定：

1. 试件破坏现象、断口方向、缩颈和滑移线等观察结果；
2. 仪器型号、接线图、装置简图和现场照片；
3. 某个通道为何无效、是否更换量程等异常说明；
4. 手工作图的取点选择以及教师指定的思考题文字；
5. 若报告只记了最终强度而未记峰值载荷，必须补录峰值载荷，不能把最终强度当作原始量。

本算例中，铸铁拉伸报告只保留了强度结果，没有记录峰值载荷。`sample_input.json` 中的 `11.05 kN` 是为演示完整流程而按报告强度反算的值，正式自动报告应改填试验机导出的原始峰值载荷。

## 三、文件说明

- `lab_report_calculator.py`：七次实验统一计算程序，只使用 Python 标准库。
- `sample_input.json`：按本次七份手写报告整理的一组完整算例。
- `sample_output/sample_result.json`：完整机器可读计算结果。
- `sample_output/sample_calculation_report.md`：可直接复制进实验报告的数据处理摘要。
- `test_calculator.py`：关键结果回归测试。

## 四、运行方法

```powershell
python lab_report_calculator.py `
  --input sample_input.json `
  --output-json sample_output/sample_result.json `
  --output-md sample_output/sample_calculation_report.md
```

输入字段缺失、读数列长度不一致或相邻级出现零增量时，程序会直接报出具体字段位置，不会静默补值。
